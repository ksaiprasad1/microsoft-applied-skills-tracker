# üìù Study Notes

This folder contains detailed study notes for each Applied Skills assessment.

## Structure

Each skill should have its own markdown file with the following format:

### File naming convention:
`skill-name.md`

### Template structure:
```markdown
# [Skill Name]

## Overview
Brief description of the skill and what it covers.

## Learning Objectives
- Objective 1
- Objective 2
- Objective 3

## Key Concepts

### Topic 1
Notes on topic 1

### Topic 2
Notes on topic 2

## Hands-on Practice
- Lab exercise 1
- Lab exercise 2

## Common Scenarios
Real-world scenarios covered in the assessment

## Tips & Tricks
- Tip 1
- Tip 2

## Resources
- Link to official docs
- Link to practice labs

## Assessment Date
Date attempted/completed

## Score/Result
Pass/Fail and notes
```

## Example Files

You can create individual note files for each skill you're studying. For example:
- `deploy-containers-azure-kubernetes.md`
- `configure-siem-microsoft-sentinel.md`
- `secure-azure-services-defender.md`
